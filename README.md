# cc

A hands-on Claude Code course. Each numbered folder is one chapter — work through them in order.

## Chapters

| # | Chapter | What you'll do |
|---|---------|----------------|
| 0 | [Onboarding](./0.onboarding/) | Set up your dev environment from scratch — Git, Node.js + npm, uv (Python), GitHub CLI — and connect your GitHub account. **Start here.** |

> More chapters are on the way.

## How to start (Chapter 0)

On a fresh machine:

1. Install Claude Code and start it (see the [official setup guide](https://code.claude.com/docs/en/setup) for macOS / Windows).
2. Tag [`0.onboarding/setup.md`](./0.onboarding/setup.md) and ask Claude Code to set up your environment.

It will install your toolchain, then walk you through logging in to GitHub — works on both macOS and Windows.
